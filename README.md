# Assignment-2
This is a C++ programme has 2 cpp files. One that calculates the execution time of 10 arrays of three different sizes having different orderings in a well formatted fashion.
And another program performs white-box testing of given code.

The following is what the programme will do:
Section_1.cpp includes:
  1) Time taken to sort randomly stored values.
  2) Time taken to sort an array having ascending ordered values.
  3) Time taken to sort an array having reverse ordered values.
  4) Sort into three different orders.

Section1.h includes one function:
  1) Sorting of numbers.

Section2.cpp includes:
  1) This file will perform white-box testing of the code and check whether the expected value is equal to actual value.

sin_cos.h includes two function:
  1) Function which is used to test the case of sin function.
  2) Function which is used to test the case of cos function.
